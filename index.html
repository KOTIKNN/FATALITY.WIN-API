Using lua scripts

1. Save the Lua script in the same folder as workspace\Fatality\lua. The file extension should be ".lua".

2. Go to the "MISC" tab. Here you'll see a list of all installed lua scripts.

3. Select the script you want to load and click "Load". "Unload" disables the script. "Reload Active Script" restarts active scripts.

Getting started

Lua scripts let you extend the cheat in various ways and are a great way to automate parts of the cheat or add that one unique feature you always wanted. To get started with Lua scripting, you'll need a suitable text editor. We recommend downloading VS Code or Sublime Text, but Notepad++ or even the built-in Microsoft Notepad will work in a pinch. If you're unfamiliar with the Lua programming language, Lua in 5 minutes is a great guide to get started.

Examples

These example show how to use certain parts of the FATALITY.WIN API and highlight best practices.

ui

Functions for interfacing with the FATALITY menu.

Fatality.ui.add_element(tabName: string, panelName: string, item: table, other: string)

Adds a UI element to a specified tab and panel in the FATALITY menu.

Argument Type Description
tabName string Name of the tab where the element will be added
panelName string Name of the panel (ScrollingFrame) within the tab
item table Configuration table for the UI element, including type and required parameters
other string Additional optional parameters (context-specific)

This function allows you to dynamically add UI elements like checkboxes, comboboxes, sliders, etc., to the FATALITY menu. The item table must include a type field (e.g., "checkbox", "combobox", "slider") and the required parameters defined in elementConfig. Supported element types include: checkbox, combobox, colorpicker, slider, bind, textentry.

Examples of adding different UI elements:

Checkbox Example:

Fatality.ui.add_element("Aimbot", "Visual", {
    type = "checkbox",
    text = "Snipline",
    var = "Snipline",
    colpicker = "SniplineCol", -- or colpicker2 = "SniplineCol2"
    bind = "SniplineBind" -- or bind2 = "SniplineBind2"
})

Combobox Example:

Fatality.ui.add_element("Aimbot", "Visual", {
    type = "combobox",
    text = "Custom ComboBox",
    var = "ComboBox",
    options = {"Option1", "Option2", "Option3"},
    moresave = true,
    colpicker = "ComboBoxCol" -- or bind = "ComboBoxBind"
})

Colorpicker Example:

Fatality.ui.add_element("Aimbot", "Visual", {
    type = "colorpicker",
    text = "Custom ColorPicker",
    var = "ColorPicker",
    colpicker = "ColorPickerCol" -- or bind = "ColorPickerBind"
})

Slider Example:

Fatality.ui.add_element("Aimbot", "Visual", {
    type = "slider",
    text = "Custom Slider",
    var = "Slider",
    min = 0,
    max = 100,
    dec = 1,
    colpicker = "SliderCol" -- or bind = "SliderBind"
})

Bind Example:

Fatality.ui.add_element("Aimbot", "Visual", {
    type = "bind",
    text = "Custom Bind",
    var = "Bind",
    colpicker = "BindCol" -- or bind = "BindBind"
})

Textentry Example:

Fatality.ui.add_element("Aimbot", "Visual", {
    type = "textentry",
    text = "Custom TextEntry",
    var = "TextEntry",
    colpicker = "TextEntryCol" -- or bind = "TextEntryBind"
})
